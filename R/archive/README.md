# Archive

This directory contains scripts that are not part of the active update workflow.

Contents include:

- older versions of preprocessing or map scripts kept for reference
- one-time validation scripts used during refactoring
- scratch scripts that were useful during development but are not sourced by `R/__RUN_UPDATES.R`

Active pipeline scripts live directly under `R/`, with shared helpers in `R/functions/` and configuration in `R/config/`.
